Etiquetas semanticas Estructurales

Es una etiqueta contenedora que no tiene semantica
header - Cabecera de un sitio o una seccion
main - Va a definir la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de pagina de un sitio web o de unas seccion
nav - Va a representar una navegacion
article - Representa una seccion auto-contenido (que por si sola se explica)
aside - Representa el contenido complementario o secundario
section - Representa una seccion de contenido
address - Representa una informacion de contacto.
